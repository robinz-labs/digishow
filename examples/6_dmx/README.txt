DMX / ArtNet Lighting Control
=============================

dmx_512.dgs
This project lists 512 DMX channels, each adjustable individually. Commonly used for DMX connection testing.

dmx_linkage.dgs
This project lists 512 DMX channels, all adjustable together. Commonly used for DMX connection testing.

artnet_512.dgs
This project lists 512 channels in ArtNet Universe 0, each adjustable individually. Commonly used for ArtNet connection testing.

artnet_linkage.dgs
This project lists 512 channels in ArtNet Universe 0, all adjustable together. Commonly used for ArtNet connection testing.

artnet_to_dmx.dgs
ArtNet-DMX Converter: maps 512 channels from ArtNet Universe 0 and 512 channels from Universe 1 to the DMX interface.

pixel_mapping
Pixel Mapping uses ArtNet / DMX channels to map pixels from images, videos, or animation files onto LED pixel strips/matrices.
This example demonstrates playing animations via Pixel Mapping and adjusting the Master brightness of the image output.
