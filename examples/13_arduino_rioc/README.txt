Arduino RIOC
============

arduino_uno.dgs
After writing the RIOC sketch to the Arduino UNO board, run this DigiShow project to perform the simplest test. Once started, the built-in LED will blink repeatedly.

aladdin_micro.dgs
After writing the RIOC sketch to the Aladdin Micro board, run this DigiShow project to perform the simplest test. Once started, the onboard output signal indicators will show a looping "flowing light" effect.
