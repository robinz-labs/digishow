SMILE FINDER
============

This demo shows how signal values are passed between a web page and DigiShow via a virtual pipe.

1. Open the project file smile_finder.dgs in DigiShow and run it

2. Open index.html in the web directory

3. When you smile at your computer’s camera, the web app detects it and sends a binary value to DigiShow, triggering the action (playing evil-laugh.wav).
