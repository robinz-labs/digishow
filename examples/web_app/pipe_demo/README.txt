PIPE DEMO
=========

This demo shows how signal values are passed between a web page and DigiShow via a virtual pipe.

1. Open the project file pipe_demo.dgs in DigiShow and run it

2. Open index.html in the web directory

3. Move the fader, toggle the switch, or click the Note button on the web page – the corresponding virtual pipe channel values in DigiShow will update in real time. 

4. Changes made in DigiShow will also update the controls on the web page.
