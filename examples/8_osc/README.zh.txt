通过 OSC 来与其他软硬件一起工作 
===========================

OSC 是一种基于 IP 网络的通信协议，DigiShow 可以透过 OSC 与外部软件交换信号变量。我们提供了一些示例：

TouchDesigner (原生支持 OSC 通信)
Unity3D (需要安装 extOSC 插件)
Blender (需要安装 NodeOSC 插件)
Python (需要安装 python-osc 库)

PythonAI
该示例演示了如何利用 AI 智能体或 vibe coding 工具生成 Python 脚本，通过 OSC 协议控制 DigiShow 中的灯光。

ArduinoESP32
该示例演示了如何让自己开发的小硬件能透过基于 WiFi 的 OSC 通信与 DigiShow 一起工作。
