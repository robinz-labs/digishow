Make Wireless Gadgets for DigiShow
==================================

If you want to make interactive gadgets that can wirelessly connect to DigiShow, you can try using an Arduino board based on the ESP32 chip (e.g., Seeed Studio XIAO ESP32). Two examples are provided here:

ble_midi
ESP32 acts as a MIDI device via Bluetooth BLE interface to wirelessly connect to DigiShow

wifi_osc
ESP32 connects wirelessly to DigiShow via WiFi, using the OSC protocol for communication

(Both the Arduino code and the DigiShow project files are provided in the examples)
